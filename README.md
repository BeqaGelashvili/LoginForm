# Crystal Glass Auth 🛡️

A modern, high-end authentication system built with pure JavaScript. Experience a sleek, professional interface with advanced glassmorphism effects right in your browser!

## 🚀 Features
* **Custom Auth Logic:** Simulated secure login and registration flow using LocalStorage.
* **Sleek UI:** Deep-dark aesthetic with animated radial glows and backdrop-blur effects.
* **Fully Responsive:** Optimized for a seamless experience across desktop, tablet, and mobile.
* **Interactive Experience:** Real-time form validation and dynamic "Logged-in" state management.
